# Nova Ženska Priča ✂️💄

Dobrodošli u GitHub repozitorijum projekta **Nova Ženska Priča** – moderni web sajt za frizerski i kozmetički salon koji kombinuje elegantan dizajn sa funkcionalnošću poput prikaza usluga, galerije radova, online prodavnice i sistema komentara i ocenjivanja proizvoda.

## 🔗 Live Demo
👉 [Pogledaj uživo](novazenskaprica.vercel.app)

## 🛠️ Tehnologije korišćene

- **React + Vite** – brza i moderna frontend arhitektura
- **Tailwind CSS** – za stilizaciju i responzivan dizajn
- **Firebase** – baza podataka za komentare i ocene proizvoda
- **React Router** – za upravljanje stranicama
- **Framer Motion** – animacije i prelazi
- **Icons (Lucide / Phosphor)** – elegantne ikone

## 🔍 Funkcionalnosti

- 💇‍♀️ Prikaz svih usluga koje salon nudi
- 🖼️ Galerija slika sa radovima
- 🛍️ Online prodavnica sa mogućnošću komentarisanja i ocenjivanja proizvoda
- 💬 Sistem komentara i recenzija (Firebase)
- 🛒 Korpa i Checkout stranica (u izradi)
- 📱 Potpuno responzivan dizajn
